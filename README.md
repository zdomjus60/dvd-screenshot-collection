# DVD Screenshot Collection

Questo repository contiene script per estrarre screenshot da video DVD e per organizzare le collezioni risultanti.

## Contenuto

*   `sample.sh`: Uno script di esempio per l'estrazione di screenshot.
*   `sample_final.sh`: Uno script finale di esempio per l'estrazione di screenshot.
*   `screenshots_final/`: Directory destinata a contenere gli screenshot finali (le immagini non sono incluse in questo repository).
*   `screenshots_hq/`: Directory destinata a contenere gli screenshot di alta qualità (le immagini non sono incluse in questo repository).

## Utilizzo degli Script

Per utilizzare gli script, posiziona il tuo file video DVD (ad esempio, `DVD1.mp4`) nella stessa directory. Gli script sono progettati per essere eseguiti da terminale.

**Esempio:**
```bash
./sample.sh
./sample_final.sh
```

**Nota:** Il file `DVD1.mp4` non è incluso in questo repository per motivi di dimensioni e privacy.
